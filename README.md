Bunny Run adalah game di mana pemain mengendalikan seekor kelinci untuk berlari dan melompat melewati rintangan hingga mencapai garis finish. Pemain menggunakan tombol A dan D untuk bergerak serta W untuk melompat. Jika karakter menyentuh rintangan, permainan akan diulang dari awal. Setelah mencapai finish, pemain harus menjawab kuis, dan jika jawaban benar maka akan muncul layar Winner sebagai tanda kemenangan.
